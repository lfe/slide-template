# slide-templates

Use this when giving LFE presentations!


## Step 0

Make sure you have `docker` installed.

## Step 1

Clone this repo:

``` shell
$ git clone git@github.com:lfe/slide-templates.git
```

## Step 2 

Run the server:

``` shell
$ make run
```

## Step 5 ... No, 3!

Edit the slides! 

Just open up `./content/_index.md` and edit as needed for your LFE presentation!
